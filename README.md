## 📖 项目介绍

欢迎来到我的 **D2L (Dive into Deep Learning)** 学习仓库

本项目用于存放我在学习 [《动手学深度学习》](https://zh.d2l.ai/) (李沐 等著) 课程过程中编写的代码、练习题解答以及相关的学习笔记。

我希望通过这个项目深入理解深度学习的底层原理，并掌握使用 PyTorch 进行模型构建与训练的能力。

## 🛠️ 环境配置

本项目主要基于以下环境：

- **操作系统**: Windows / Linux
- **编程语言**: Python 3.x
- **深度学习框架**: PyTorch
- **IDE**: VS Code / Jupyter Notebook
